# Economicon

## 1. コンセプト

**「計量経済学を、もっと身近に、直感的に。」**
Economiconは、データ読み込みから高度な統計分析（固定効果、操作変数法など）までをGUIで完結させる、学習者・研究者・アナリストのための「分析サンドボックス」です。

- **直感的**: コード不要。クリックだけでモデルを構築し、結果を可視化。
- **学習重視**: 分析の作法（検定の順序など）をナビゲート。
- **信頼の透明性**: 内部ロジックは Python の標準ライブラリに基づき、R との整合性を継続的にテスト。

## 3. バリデーション（信頼性の証明）

本プロジェクトでは、統計学的な妥当性を維持するために以下の二重テストを実施しています。

- **Pattern A**: 内部処理が Python の `statsmodels` / `linearmodels` の出力と一致することの検証。
- **Pattern B**: 標準的なデータセット（Grunfeld 等）を用い、R の `plm` / `fixest` パッケージの算出結果と $10^{-8}$ 以下の誤差で一致することの検証。
